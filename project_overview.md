# 1. Project Overview

## 🎯 Vision

Build a **Goal-Driven Visual AI Copilot** that:

1. Accepts a user-defined goal (e.g., “Create a GitHub repository”)
2. Uses an LLM to generate a structured step-by-step plan
3. Enters a **Guidance Mode**
4. Monitors the user’s screen (via screen share)
5. Visually overlays hints (like a game tutorial)
6. Guides the user step-by-step safely (no auto-clicking)

The assistant is:
- Human-in-the-loop
- Visual-first
- Structured
- Safe
- Goal-driven

This is NOT a chatbot.
This is an AI workflow navigation system.

# 4. High Level Workflow
## Start with Chat mode (normal mode)
1. User enter the goal.
2. System, specifically the LLM（chat model） generates structured step plan
3. User confirm the step generated by the chat model
## Continue with the guidance mode (screen capture mode)
4. User elects to enter Guidance Mode, then manually copies the plan generated in Chat Mode and pastes it into the Guidance Mode prompt input
5. User share the screen (browser page)
6. Frontend captures a screenshot at a user-configurable interval (default: 5 seconds) and sends it to the backend
7. Backend Send screenshot to the multi-modal LLM (the model that can analyze image)
8. The multi-modal LLM returns:
    + Current step
    + UI hint configuration for current step
    + The Next step

    to the backend
9. Frontend display instructions overlay on user screen
10. Repeat until reach the final step, or the user explicitly end
